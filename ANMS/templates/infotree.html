Petunjuk membaca tree adalah sebagai berikut :

  • Node akar adalah IPK (mempunyai pengaruh paling tinggi pada tree)
  • Node kedua dari node akar/node atribut adalah nilai dari atribut yang bersangkutan
  • Node cabang adalah node selanjutnya dari node akar/node cabang sebelumnya, node cabang mempunyai 4 atribut (jurusan, jenis kelamin, TOEFL, asal)
  • Node terakhir dari setiap jalur merupakan kelas/target yaitu Tepat dan Telat
